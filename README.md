## 👨‍💻 About Me

**B.Tech in AI & Data Science** | Kurukshetra University

I build machine learning systems from data to deployment. My work focuses on **production-ready pipelines**, **agentic AI**, and understanding how models behave in the real world — not just their accuracy.

> *I prefer building and experimenting over treating models as black boxes.*

**What I build:** End-to-end ML pipelines • RAG systems • Agentic workflows • Production deployments • Observability systems

## 🚀 Featured Projects

### 🤖 [Agentic Research Assistant](https://github.com/ayushsyntax/Agentic-Research-Assistant)
Production agentic AI with LangGraph, tool-calling, persistent memory, and RAG. Full observability via LangSmith.  
`LangGraph` · `ChromaDB` · `LangSmith` · `FastAPI` · `Streamlit`

### 🏠 [End-to-End ML Pipeline](https://github.com/ayushsyntax/Regression_ML-End-to-End)
Complete production system for housing prediction with time-series validation, MLflow tracking, Docker deployment, and CI/CD.  
`XGBoost` · `Optuna` · `MLflow` · `Docker` · `AWS ECS`

### 🧬 [GPT from Scratch](https://github.com/ayushsyntax/GPT-from-Scratch-in-PyTorch)
6-layer transformer built from first principles. Trained on Shakespeare (1.1M chars). Perplexity: 3.02 • Accuracy: 64.9%  
`PyTorch` · `NumPy` · **[Live Demo →](https://huggingface.co/spaces/ayushsyntax/shakespeare-gpt)**

### 🔄 [Reasona](https://github.com/ayushsyntax/Reasona)
Self-improving RAG system using HyDE + SEAL architecture with critic-based feedback loops.  
`LangChain` · `ChromaDB` · `Ollama`
